# Como conciliar extratos bancários

A conciliação bancária é o processo de comparar as transações do seu extrato com as contas a pagar/receber cadastradas no sistema.

## 1. Importar o extrato

1. Acesse **Finanças > Conciliação** no menu lateral
2. Clique em **Importar extrato**
3. Selecione o arquivo **.OFX** do seu banco
4. Aguarde o processamento

## 2. Revisar as sugestões

Após a importação, o sistema exibirá:

- **Conciliadas automaticamente** (verde): a IA encontrou correspondência exata
- **Sugestões de correspondência** (amarelo): o sistema sugeriu uma possível correspondência
- **Sem correspondência** (vermelho): necessita de ação manual

## 3. Conciliar manualmente

Para transações sem correspondência automática:

1. Clique na transação do extrato
2. Busque a conta a pagar/receber correspondente
3. Clique em **Vincular**

## 4. Finalizar conciliação

Após revisar todas as transações:

1. Clique em **Finalizar conciliação**
2. Confirme o período
3. O relatório estará disponível em **Relatórios**

## Dicas para melhorar a conciliação automática

- Cadastre as contas a pagar/receber com valores exatos e datas corretas
- Use descrições padronizadas nas transferências
- Configure o Tiny ERP para que pedidos entrem automaticamente
