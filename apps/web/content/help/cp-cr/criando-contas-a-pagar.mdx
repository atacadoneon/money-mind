# Criando contas a pagar

## Acessar a tela de contas a pagar

1. No menu lateral, clique em **Finanças**
2. Selecione **Contas a Pagar**

## Criar uma nova conta

1. Clique no botão **+ Nova conta**
2. Preencha os campos obrigatórios:
   - **Descrição** — nome do fornecedor ou despesa
   - **Valor** — valor a pagar
   - **Data de vencimento** — quando vence
3. Campos opcionais:
   - **Fornecedor** — vincule a um contato cadastrado
   - **Categoria** — para relatórios por categoria
   - **Centro de custo** — empresa específica
   - **Observação** — notas internas
4. Clique em **Salvar**

## Status das contas

- **Em aberto** — ainda não pago, dentro do prazo
- **Vencido** — prazo expirado sem pagamento
- **Pago** — baixado com registro de pagamento

## Dar baixa em uma conta

1. Localize a conta na lista
2. Clique em **Dar baixa**
3. Confirme a data de pagamento e o valor pago
4. Selecione a conta bancária debitada

## Dicas

- Use categorias consistentes para relatórios precisos
- Configure alertas de vencimento em **Configurações**
- Importe em lote via planilha Excel/CSV para cadastros massivos
